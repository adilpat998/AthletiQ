## User Stories

### 1. Workout Plan Management (CRUD)

#### Creating a Workout Plan
**As a** user,  
**I want** to create a workout plan tailored to my fitness goal,  
**So that** I can follow a structured program to achieve my desired results.  

**Acceptance Criteria:**  
- Users must provide their goal, title, description, and training days per week.   
- A structured week-to-week workout plan is saved in JSON format.  

#### Viewing a Workout Plan
**As a** user,  
**I want** to view my saved workout plans,  
**So that** I can track my fitness progress.  

**Acceptance Criteria:**  
- Users can see a list of all their workout plans.  
- Users can view details such as exercises, training days, and goals.  

#### Updating a Workout Plan
**As a** user,  
**I want** to update my workout plan,  
**So that** I can adjust it based on my progress.  
 
- Users must be able to edit any part of their workout plan.  
- Updates should be saved and reflected immediately.  

#### Deleting a Workout Plan
**As a** user,  
**I want** to delete my workout plan,  
**So that** I can remove outdated or unnecessary plans.  

**Acceptance Criteria:**  
- Users must be able to delete a workout plan.  
- A confirmation prompt should appear before deletion.  

---

### 2. Social Feed (Instagram-Like Post System)

#### Creating a Post
**As a** user,  
**I want** to create a post about my fitness progress or workout log,  
**So that** I can share my journey with others.  

**Acceptance Criteria:**  
- Users must be able to create a post with text and images.  
- Posts should be displayed on the social feed.  

#### Viewing Posts
**As a** user,  
**I want** to view posts from other users,  
**So that** I can stay motivated and engage with the community.  

**Acceptance Criteria:**  
- Users can browse posts made by others.  
- Posts should display the author, content, and timestamp.  

#### Updating a Post
**As a** user,  
**I want** to update my post,  
**So that** I can edit or correct information.  

**Acceptance Criteria:**  
- Users must be able to edit the text of their posts.  
- Updates should be saved and displayed immediately.  

#### Deleting a Post
**As a** user,  
**I want** to delete my post,  
**So that** I can remove content I no longer want to share.  

**Acceptance Criteria:**  
- Users must be able to delete their own posts.  
- A confirmation prompt should appear before deletion.  

---

### 3. Reviews for Workout Plans

#### Creating a Review
**As a** user,  
**I want** to leave a review for a workout plan,  
**So that** I can share my experience and help others decide.  

**Acceptance Criteria:**  
- Users must be able to write a review for a workout plan.  
- Users can rate the plan from 1 to 5 stars.  
- Reviews should include comments and feedback.  

#### Viewing Reviews
**As a** user,  
**I want** to read reviews from others,  
**So that** I can make an informed decision before using a workout plan.  

**Acceptance Criteria:**  
- Users can see all reviews for a specific workout plan.  
- Reviews should display rating, comments, and reviewer name.  

#### Deleting a Review
**As a** user,  
**I want** to delete my own review,  
**So that** I can remove feedback that is no longer relevant.  

**Acceptance Criteria:**  
- Users must be able to delete their own reviews.  
- A confirmation prompt should appear before deletion.  

---

### 4. Comments on Instagram-Like Posts

#### Creating a Comment
**As a** user,  
**I want** to comment on posts,  
**So that** I can interact with other users.  

**Acceptance Criteria:**  
- Users must be able to leave comments on posts.  
- Comments should be displayed under the respective posts.  

#### Viewing Comments
**As a** user,  
**I want** to read comments on posts,  
**So that** I can see what others are saying.  

**Acceptance Criteria:**  
- Users must be able to see all comments on a post.  
- Comments should display the author, text, and timestamp.  

#### Deleting a Comment
**As a** user,  
**I want** to delete my own comments,  
**So that** I can remove messages I no longer want visible.  

**Acceptance Criteria:**  
- Users must be able to delete their own comments.  
- A confirmation prompt should appear before deletion.
